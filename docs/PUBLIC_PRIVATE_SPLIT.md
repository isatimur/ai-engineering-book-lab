# Public / Private Split

## Safe to publish after review
- process docs
- agent instructions
- research pass logs
- claims ledgers
- evidence packs
- selected synthesis pages
- selected manuscript artifacts
- scripts that do not contain secrets or machine-specific assumptions

## Keep local unless curated
- raw transcript caches
- bulk temporary inventories
- local absolute paths
- machine-specific helper scripts
- rough working notes not meant for public view

## Review rule
Nothing becomes public just because it exists locally. Promote intentionally.
