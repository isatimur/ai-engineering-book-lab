# Chapter Packets v1

These packets turn the current outline into chapter-level working files.

## Packet structure
Each chapter file contains:
- chapter role in the book
- supporting source cluster
- strongest source-backed claims
- useful quotes/excerpts to revisit when drafting
- open questions requiring editorial judgment

## Reproducibility note
Claims and excerpts were synthesized from the current processed corpus, primarily:
- the chapter outline and synthesis pack in `05_Book_Ideas/`
- source note summaries and transcript excerpts in `01_Videos/`
- concept syntheses in `04_Concepts/`

These are drafting packets, not final prose. They are designed so a human editor can trace each chapter back to concrete talks and then deepen specific citations during manuscript writing.
