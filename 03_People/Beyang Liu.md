## Overview

Beyang Liu co-founded Sourcegraph, the enterprise code intelligence platform, and has led its evolution into AI-native coding agent territory through Amp Code. He is one of the most technically credible voices on what it takes to build coding agents that work across large, real-world codebases rather than toy projects.

## Key claims / positions

- Coding agents need whole-repository context, not file-level context — code intelligence (graph traversal, symbol resolution) is what separates useful agents from autocomplete.
- The "emerging skillset" for engineers is task decomposition and agent direction, not prompt engineering for line completions.
- AI coding tools succeed or fail based on code indexing quality, not model quality alone.
- The IDE is not the right unit of abstraction for agents that need to reason about multi-file, multi-repo changes.
