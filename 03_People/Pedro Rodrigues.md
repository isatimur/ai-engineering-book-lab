## Overview

Pedro Rodrigues works at Supabase on developer experience and AI integration. His AI Engineer appearances focus on how MCP and "skills" (higher-level, composable agent capabilities) can close the context gap between what agents know and what they need to know to act correctly on real databases and infrastructure.

## Key claims / positions

- The "context gap" is the distance between the agent's general knowledge and the specific runtime state it needs to act correctly.
- MCP tools provide access; skills provide context — both are needed, neither alone is sufficient.
- Supabase's approach: MCP for database access, skills for encoding best practices (migration patterns, RLS policies) the agent should follow.
- Composable skills reduce the prompt engineering burden on the developer and make agents more reliable across different user contexts.
