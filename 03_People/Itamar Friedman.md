## Overview

Itamar Friedman leads Qodo, a company focused on AI-powered code integrity — moving beyond code generation to code quality, test generation, and review automation. A prominent voice on what "AI-generated code quality" actually means in practice.

## Key claims / positions

- AI code generation volume has outpaced code quality verification; the bottleneck is now trust, not output.
- "Vibe coding" (prompt-driven development without review) is risky without a quality gate layer.
- Test generation is a better near-term AI ROI than code generation alone, because it makes AI-generated code trustworthy.
- Code review automation requires richer context than a single file diff — whole-repo graph understanding matters.
