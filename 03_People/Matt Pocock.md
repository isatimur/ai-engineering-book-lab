## Overview

Matt Pocock is the creator of Total TypeScript and one of the most prominent TypeScript educators in the web development community. His AI Engineer appearances focus on how TypeScript's type system remains essential — arguably becomes more essential — as AI generates more code, because types are the cheapest form of automated verification.

## Key claims / positions

- Software fundamentals (types, tests, clean interfaces) become more important with AI code generation, not less.
- AI generates syntactically valid code that fails type checks — TypeScript's type system is a free correctness layer.
- The developer who understands what the AI generated is more valuable than the developer who generated it fastest.
- "Vibe coding" without typed interfaces is technical debt that compounds at AI speed.
