## Overview

Tomas Reimers co-founded Graphite, a code review and stacked diff platform. His AI Engineer talks present empirical data from millions of AI code reviews, offering one of the most grounded perspectives on what AI code review actually catches vs. misses at scale.

## Key claims / positions

- At scale, AI code review catches different categories of bugs than human review — they are complementary, not substitutes.
- "One-shotting" (generating and merging code without review) creates technical debt that compounds quickly; AI review is cheap insurance.
- The merge pipeline (test → review → merge → deploy) is the correct unit of AI automation, not just generation.
- Most AI code review tools are tuned for false-negative minimization (don't miss bugs) but false-positive rate kills adoption.
