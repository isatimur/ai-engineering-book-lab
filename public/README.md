# Public Content Layer

This folder contains the first curated public-safe content mirrored from the larger local working vault.

Selection principles:
- useful without requiring the full private/local corpus
- low risk of leaking machine-specific or transcript-heavy artifacts
- representative of the project's actual book/research work
- readable by outside contributors and future agents

Current contents:
- selected concept synthesis pages
- selected draft-ready chapter starter material

This is intentionally curated rather than exhaustive.
